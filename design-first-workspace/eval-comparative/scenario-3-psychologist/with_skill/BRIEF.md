# Design Brief: Personal Website for Psychologist Anna Svetlova

## Project Summary
Personal website for psychologist Anna Svetlova specializing in anxiety and burnout. Primary goal: appointment booking via form or messenger CTA.

## Target Audience
- Age: 25-40
- Sources: recommendations, search engines
- Devices: mixed mobile/desktop

## Emotional Goals
- Safety and trust
- Calm professionalism
- Warmth without clinical coldness

## Primary Action
Book a consultation (form submission or CTA leading to messenger)

## Specialization
Anxiety and burnout therapy

## Brand
- Name: Anna Svetlova (Анна Светлова)
- No existing brand identity
- Tone: professional but approachable

## Content Requirements
- Photo placeholder (abstract/geometric — no real photo)
- About section
- Approach description (bullets)
- Booking CTA

## Anti-References
- NOT cold medical/clinical
- NOT esoteric/mystical
- NOT overly corporate

## Technical Constraints
- Vanilla HTML with inline CSS
- Google Fonts CDN only (Cyrillic-safe)
- Mobile-first responsive design
- Russian language content

## Prototype Directions (6 variants)

| ID | Direction | Key Traits |
|----|-----------|------------|
| A | Soft Neutral | Cream/beige, elegant serif, centered, airy whitespace |
| B | Modern Minimal | White bg, thin sans-serif, accent line, editorial feel |
| C | Warm Sage | Sage green palette, split layout, rounded photo placeholder |
| D | Twilight Calm | Muted blue-purple, gradient, glassmorphism elements |
| E | Earth & Paper | Paper texture feel, warm browns, serif, editorial columns |
| F | Clean Clinical-Warm | White + warm coral accent, structured grid, professional |
