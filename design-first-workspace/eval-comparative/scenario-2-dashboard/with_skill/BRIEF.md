# MetricFlow — Design Brief

## Project
SaaS analytics dashboard for startup metrics (MRR, churn, LTV, active users).

## Audience
Founders and product managers, 25-40. Desktop-primary, mobile-capable.

## Emotional Goals
Professional confidence, data clarity, sense of control, modern tech aesthetic.

## Primary Actions
- Understand metrics at a glance
- Drill into data trends

## Brand
"MetricFlow" — modern SaaS, no existing brand guidelines. Clean, trustworthy, data-forward.

## Content Inventory
| Metric | Value |
|--------|-------|
| MRR | $48.2K |
| Churn | 3.2% |
| LTV | $1,240 |
| Active Users | 2,847 |

Charts: trend lines, bar charts (mock via CSS/SVG).

## Anti-References
- No cluttered enterprise dashboards
- No boring corporate gray
- No dense tables without visual hierarchy

## Tech Constraints
- Vanilla HTML + CSS only (no JS libraries)
- Inline SVG or CSS-only charts
- Google Fonts CDN (Cyrillic-safe only)
- Mobile-first responsive
