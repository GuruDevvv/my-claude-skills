# my-claude-skills

My custom skills for [Claude Code](https://claude.ai/claude-code).

Each skill is a folder with a `SKILL.md` file — drop it into `~/.claude/skills/<skill-name>/` and restart Claude Code.

---

## Skills

| Skill | Description |
|-------|-------------|
| [multi-layer-review](multi-layer-review/) | Run your spec through a chain of up to 5 reviewers before coding — architecture, code, user POV, robustness, requirements |
